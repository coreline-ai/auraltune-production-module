3. Quick Start

мғҲ мҳӨл””мҳӨ нҢҢмқҙн”„лқјмқём—җм„ң м—”м§„мқ„ л¶ҷмқҙлҠ” мөңмҶҢ нқҗлҰ„мһ…лӢҲлӢӨ.

import com.coreline.audio.AudioEngine
import com.coreline.audio.Builder
import java.nio.ByteBuffer
import java.nio.ByteOrder

// 1. мӢӨм ң PCM stream sampleRateмҷҖ к°ҷмқҖ к°’мңјлЎң м—”м§„ мғқм„ұ
val engine = AudioEngine.Builder()
    .forSampleRate(48_000)
    .build()

try {
    // 2. н•„мҡ”н•ҳл©ҙ EQ м„Өм •
    engine.updateManualEq(
        frequencies = floatArrayOf(80f, 2500f, 7000f),
        gainsDB = floatArrayOf(3.0f, 2.0f, -1.5f),
        qFactors = floatArrayOf(0.7f, 1.2f, 2.5f),
        filterTypes = intArrayOf(1, 0, 0), // LOW_SHELF, PEAKING, PEAKING
    )
    engine.setManualEqEnabled(true)

    // 3. мҳӨл””мҳӨ мҪңл°ұ/AudioProcessorм—җм„ң stereo interleaved float32 мІҳлҰ¬
    val frames = 1024
    val pcm = ByteBuffer
        .allocateDirect(frames * 2 * 4)
        .order(ByteOrder.nativeOrder())

    val rc = engine.process(pcm, frames)
    check(rc == 0) { "engine.process failed: $rc" }
} finally {
    // 4. мҳӨл””мҳӨ мҠӨл Ҳл“ңк°Җ лҒқлӮң л’Ө н•ҙм ң
    engine.close()
}
мӨ‘мҡ”: м—”м§„мқҳ sampleRateлҠ” мӢӨм ңлЎң мІҳлҰ¬н•ҳлҠ” PCM stream rateмҷҖ к°ҷм•„м•ј н•©лӢҲлӢӨ. 44.1kHz мқҢмӣҗмқ„ 48kHz м—”м§„мңјлЎң мІҳлҰ¬н•ҳл©ҙ мҳӨл””мҳӨлҠ” нқҗлҘҙм§Җл§Ң н•„н„° мӨ‘мӢ¬ мЈјнҢҢмҲҳк°Җ л№„мңЁл§ҢнҒј м–ҙкёӢлӮ©лӢҲлӢӨ.

5.2 AutoEQ / OPRA н”„лЎңнҢҢмқј м Ғмҡ©

AutoEQмҷҖ OPRA лӘЁл‘җ native м—”м§„ мһ…мһҘм—җм„ңлҠ” к°ҷмқҖ AutoEQ filter chainмһ…лӢҲлӢӨ. м°ЁмқҙлҠ” лҚ°мқҙн„° м¶ңмІҳмҷҖ нҢҢмӢұ кІҪлЎңмһ…лӢҲлӢӨ.

import com.coreline.audio.EqFilterType

val types = intArrayOf(
    EqFilterType.PEAKING.nativeId,
    EqFilterType.LOW_SHELF.nativeId,
    EqFilterType.HIGH_SHELF.nativeId,
)
val freqs = floatArrayOf(1200f, 80f, 9000f)
val gains = floatArrayOf(-2.0f, 3.5f, -1.5f)
val qs = floatArrayOf(1.2f, 0.7f, 0.8f)

val rc = engine.updateAutoEq(
    preampDB = -5.5f,
    enableLimiter = true,
    profileOptimizedRate = 48_000.0,
    filterTypes = types,
    frequencies = freqs,
    gainsDB = gains,
    qFactors = qs,
)
check(rc == 0) { "AutoEQ rejected: $rc" }

// мқјл°ҳ м „нҷҳ: 0.5мҙҲ wet/dry fade
engine.setAutoEqEnabled(true, immediate = false)

// кёҙкёү bypass: fade м—Ҷмқҙ мҰүмӢң off
engine.setAutoEqEnabled(false, immediate = true)

6. мҳӨл””мҳӨ мІҳлҰ¬ мғҳн”Ң

6.1 нҳ„мһ¬ м•ұ л°©мӢқ: Float32 in-place мІҳлҰ¬

нҳ„мһ¬ AuralTune м•ұмқҖ мқҙ кІҪлЎңлҘј мӮ¬мҡ©н•©лӢҲлӢӨ. мһ…л Ҙ PCMмқ„ Kotlinм—җм„ң float32 direct bufferлЎң ліҖнҷҳн•ҳкі , native м—”м§„мқҖ float32 stereo interleaved bufferлҘј in-placeлЎң мІҳлҰ¬н•©лӢҲлӢӨ.

val channels = 2
val frames = 1024
val samples = frames * channels

val floatBuffer = ByteBuffer
    .allocateDirect(samples * 4)
    .order(ByteOrder.nativeOrder())

// floatBufferм—җ stereo interleaved float32 PCMмқ„ мұ„мӣҖ
// L0, R0, L1, R1, ...

val rc = engine.process(floatBuffer, frames)
if (rc != 0) {
    // мӢӨмӢңк°„ мҪңл°ұм—җм„ңлҠ” throw лҢҖмӢ  bypass/fallback к¶ҢмһҘ
}

6.2 кі н•ҙмғҒлҸ„ нҷ•мһҘ л°©мӢқ: 24-bit packed мһ…м¶ңл Ҙ

processFormatted()лҠ” SO лӮҙл¶Җм—җм„ң S24 вҶ’ float32 вҶ’ DSP вҶ’ S24 ліҖнҷҳмқ„ мҲҳн–үн•©лӢҲлӢӨ.

import com.coreline.audio.PcmFormat
import java.nio.ByteBuffer
import java.nio.ByteOrder

val frames = 1024
val channels = 2

val inputS24 = ByteBuffer
    .allocateDirect(frames * channels * PcmFormat.S24.bytesPerSample)
    .order(ByteOrder.nativeOrder())

val outputS24 = ByteBuffer
    .allocateDirect(frames * channels * PcmFormat.S24.bytesPerSample)
    .order(ByteOrder.nativeOrder())

// inputS24м—җ little-endian 24-bit packed PCM мұ„мҡ°кё°

val rc = engine.processFormatted(
    input = inputS24,
    inputFormat = PcmFormat.S24,
    output = outputS24,
    outputFormat = PcmFormat.S24,
    numFrames = frames,
)
check(rc == 0) { "processFormatted failed: $rc" }

// outputS24лҘј ENCODING_PCM_24BIT_PACKED AudioTrackм—җ write
